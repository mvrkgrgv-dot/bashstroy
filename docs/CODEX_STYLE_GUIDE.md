# CODEX Style Guide

## 1. Общий стиль

- Светлый премиальный сайт.
- Белый, молочный, ivory, warm beige.
- Никаких тёмных тяжёлых секций.
- Много воздуха.
- Мягкие скругления.
- Деликатный liquid glass.
- Тонкие линии.
- Лёгкие тени.
- Премиальный архитектурный характер.

## 2. Типографика

- Для заголовков использовать элегантный editorial serif.
- Подключить через Google Fonts один из вариантов: Cormorant Garamond / Playfair Display / Lora.
- Для текста и интерфейса использовать Inter или Manrope.
- Заголовки должны быть крупными, воздушными, с красивым line-height.
- Не использовать грубые жирные шрифты.
- Не делать сайт похожим на шаблон.

## 3. Анимации

- Использовать только осмысленные анимации.
- UI-анимации 120-250ms.
- Не использовать `transition: all`.
- Использовать `--ease-out: cubic-bezier(0.23, 1, 0.32, 1);`.
- Использовать `--ease-in-out: cubic-bezier(0.77, 0, 0.175, 1);`.
- Кнопки при клике: `transform: scale(0.97)`.
- Элементы при появлении: `opacity + translateY(16px)`, не `scale(0)`.
- Liquid glass должен быть деликатным.

## 4. Компоненты

- Header, CTA, feature strip, project cards, footer должны быть аккуратными.
- Все карточки с `border-radius` 24-36px.
- Большие секции можно скруглять до 40px.
- Эффект glass использовать выборочно, не везде.

## 5. Цель

Сайт должен запоминаться посетителю как чистый, дорогой, свежий и уверенный архитектурный бренд.

После создания файла всегда сверяй дальнейшие изменения с ним.
